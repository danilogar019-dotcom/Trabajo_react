## 📖 Description

**Wanderlust Travels** is a premium Single Page Application (SPA) built with React.js. It offers an immersive interface for discovering travel destinations, sharing experiences in a real-time community forum, and managing travel inquiries.

The platform focuses on **visual excellence**, **fluid responsiveness**, and **clean code architecture**.

### Key Features
*   **Immersive Home Page**: Accessible via `/` and `/home`, featuring animated statistics and a vibrant hero section.
*   **Real-time Travelers Forum**: A community hub integrated with **Firebase Firestore** for authentic experience sharing with category filtering and live search.
*   **Interactive Destinations Grid**: A responsive layout using **Flexbox** and **Media Queries** to showcase global travel packages with smooth hover-tilt effects.
*   **RSS Integration**: A dedicated page providing syndication information for travel enthusiasts.
*   **Multi-step Booking**: A user-centric wizard for secure and intuitive travel reservations.

## Firebase hosting
Wanderlust Travels is deployed using Firebase Hosting, ensuring fast global delivery, SSL security, and seamless SPA routing.

Live Deployment:  
https://basewanderland.web.app

Deployment Workflow:

Built with Vite using npm run build

Hosted from the optimized dist/ directory

Configured as a Single Page Application (SPA)

Firebase handles URL rewrites to support React Router navigation

Firebase Tools Used:

Firebase CLI for deployment

Hosting configuration via firebase.json

Automatic SPA routing with rewrites
